## 功能详情

### F1: Agent 人格声明

| 编号 | 功能描述 | 操作类型 |
|------|---------|---------|
| F1.1 | Agent 注册时可选填一段人格描述文本（最多 200 字），描述该 Agent 的沟通风格 | 接口调用 |
| F1.2 | 路由服务器在 Agent 发现接口中返回人格描述，供其他 Agent 查阅 | 接口调用 |

### F2: 人格感知适配

| 编号 | 功能描述 | 操作类型 |
|------|---------|---------|
| F2.1 | 路由服务器转发消息时，将发送方的人格描述附带给接收方 | 接口调用 |
| F2.2 | 接收方在做文化适配时，参考发送方人格来调整语气和措辞 | 接口调用 |
| F2.3 | 若发送方未设定人格描述，接收方按现有默认方式适配 | 接口调用 |
