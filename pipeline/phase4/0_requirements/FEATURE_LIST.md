## 功能详情

### F1: Agent 注册鉴权

| 编号 | 功能描述 | 操作类型 |
|------|---------|---------|
| F1.1 | 路由服务器启动时从环境变量加载 API key 列表 | 接口调用 |
| F1.2 | POST /agents、POST /messages、DELETE /agents/:id 需携带 Bearer token | 接口调用 |
| F1.3 | 请求无 token 或 token 无效时返回 401 错误 | 接口调用 |

### F2: 健康检查

| 编号 | 功能描述 | 操作类型 |
|------|---------|---------|
| F2.1 | GET /health 返回服务状态（无需鉴权） | 接口调用 |

### F3: 环境变量配置

| 编号 | 功能描述 | 操作类型 |
|------|---------|---------|
| F3.1 | 端口(PORT)、鉴权密钥(CHORUS_API_KEYS)通过环境变量注入 | 接口调用 |
| F3.2 | 服务器入口读取环境变量替代硬编码值 | 接口调用 |

### F4: 容器化

| 编号 | 功能描述 | 操作类型 |
|------|---------|---------|
| F4.1 | 提供 Dockerfile 将路由服务器打包为可运行镜像 | 接口调用 |
| F4.2 | 提供 .dockerignore 排除 node_modules、tests、spike 等 | 接口调用 |
